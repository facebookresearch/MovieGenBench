# MovieGenBench

We introduce our evaluation benchmark "Movie Gen Bench" for Movie Gen project, which includes Movie Gen Video Bench (Section 3.5.2) and Movie Gen Audio Bench (Section 6.3.2) This repository contains the 1) 1000 text-to-video testing prompts and a link to model outputs on those prompts 2) Ann to fill in for audio.
